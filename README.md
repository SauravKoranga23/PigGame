# Pig Game 🎲

A simple two-player dice game built with JavaScript.

## Features
- Two player gameplay
- Roll dice functionality
- Hold score
- Player switching logic
- Win condition

## Tech Used
- HTML
- CSS
- JavaScript

## Demo
Play until a player reaches 30 points. But the catch is you will lose all your points if you roll the number 1 on the dice 🎲
